Description Lists

A description list <dl> is used to create a description list with terms and definitions.

Description Terms

A description term <dt> is the list name given to each part of the list.

Description Definition

A description definition <dd>is used to define and describe each term in your description list.

Here is an example of dl, dd and dt with code and no code

<dl>
<dt>Description List</dt>
<dd> Defines a list of descriptions</dd>
<dt>Description Term</dt>
<dd>Defines a term in the list</dd>
<dt>Description Definition</dt>
<dd>Defines the term given to the list</dd>
</dl>
Description List
Defines a list
Description Term
Defines a term in list
Description Defintion
Defines the description for the term in list

Links

The <a> tag is used to create links that connect to other webpages or files.
A tags must have the href attribute in order to create links to work otherwise it won't know where to go.


Here is an example of an a tag that links to google.
<a href="https://www.google.com">google.com</a>

Semantics

Semantic tags are tags that basically do nothing to affect the website but can be used as containers for your webpage so it could be styled better for later.

List of Semantics
<div> Defines a divider as a container
<span> Defines an inline container for representing and marking up content
<nav> Defines a set of navigation links as a container
<section> Defines a section in a webpage used as a container
<article> Defines an article used in webpage as a containter
<header> Defines header used as a container to introduce your webpage or section
<footer> Defines footer used as a container to conclude your webpage or section
<aside> Defines content that is set aside of a webpage used as a sidebar

Images

In order to create images we use the img tag with the src attribute and for accessability we use the alt tag. You can also adjust the width and height of the image using the width and height attributes.

img
creates an image
src
defines the source of the image
alt
used for accessability to identify the image to a blind or deaf person
width
adjust the width of the image
height
adjust the height of an image
